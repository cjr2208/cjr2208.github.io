Blog for Sound of (College) Life

This was super fun to program! Firstly, I had no idea how to build a grid that was interactable; I thought I was going to have to use 100 buttons, which I think is what the former student did in the example you showed us, but I thought that would be too many variables and I honestly wasn’t sure how I could set it up.

I had the idea to do the videos kinda like the other student example you showed us in class with the dog, so I knew I wouldn’t be able to do that with buttons. Thus, I ended up looking up ways to create an interactable grid of cells and I found on a website called W3 schools that you could make a table and then add event listeners to the cells in that table.

Once I got the table set up, doing the game of life wasn’t very difficult; there are so many tutorials online for how to do it, but I was really worried about adding the audio.

I had to mess around with the frequencies of the notes and location. I found the frequencies I wanted to use on a website, but then I played around a lot with the location and ended up just making it linearly increase which I know is boring, but I tried to randomize it or do different quadrants and it just didn’t sound good.

I finally learned a much more efficient way to avoid clipping which I was worried about since the number of cells changes all the time, and I definitely should've used this method originally in my HW1 and 2. Doing the videos was also super new for me. I knew how to have a video on a website with html, but I had no idea how I would get them to turn on and off with the cells being activated. It wasn’t horribly difficult though, it made sense to just set them up similarly to the actual cells, but I did have to learn how to control and pause videos which I also got from W3schools!

I really really loved this assignment; it is actually making me think about doing something similar for my final project, and I hope you enjoy, but please lol give the website a bit of time to load!!
