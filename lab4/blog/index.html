Blog for Sound of (Dog) Life

This was super fun to program! Firstly, I had no idea how to build a grid that was interactable; I thought I was going to have to use 100 buttons, which I think is what the former student did in the example you showed us, but I thought that would be too many variables and I honestly wasn’t sure how I could set it up.

I had the idea to do the videos kinda like the other student example you showed us in class with the dog, so I knew I wouldn’t be able to do that with buttons. Thus, I ended up looking up ways to create an interactable grid of cells and I found on a website called W3 schools that you could make a table and then add event listeners to the cells in that table.

Once I got the table set up, doing the game of life wasn’t very difficult; there are so many tutorials online for how to do it, but I was really worried about adding the audio.

I had to mess around with the frequencies of the notes and location of each video. I found the frequencies I wanted to use for each clip by using this cool app called anthem score which basically takes a video or audio clip and shows the notes playing (I used this before when I was practicing live coding).

For the location of each video, I thought about doing the frequencies linearly, then random, and then quadrants based on scale, but I felt like the random sounded the most diverse, since the game of life often gets stuck in a certain area for long periods of time.

I finally learned a much more efficient way to avoid clipping which I was worried about since the number of cells changes all the time, and I definitely should've used this method originally in my HW1 and 2. Doing the videos was also super new for me. I knew how to have a video on a website with html, but I had no idea how I would get them to turn on and off with the cells being activated. It wasn’t horribly difficult though, it made sense to just set them up similarly to the actual cells, but I did have to learn how to control and pause videos which I also got from W3schools! BY FARRRR the worst part was uploading the videos to github omg you have no idea how long that took. They don't let you have videos over a certain amount of mb and they don't let you upload more than like five videos at a time. I HAD 100. Horrible decision. I'm not joking I spent two hours trying to upload videos only to get 50 in and realize that some of them just don't show up for some reason. So I scrapped the multiple videos idea and went with my dog, but in the end I think it looks nice.

I really really loved this assignment; it is actually making me think about doing something similar for my final project, and I hope you enjoy, but please lol give the website a bit of time to load!!
